## Basic Usage

```jsx
import { useQuery } from './utils/lite-query';

const { data, error, isLoading } = useQuery({
	queryKey: 'posts',
	queryFn: () => fetch('https://jsonplaceholder.typicode.com/posts').then(res => res.json())
});

if (isLoading) return 'Loading...';
if (error) return `Error: ${error.message}`;
```

## How to Install

1. Simply copy [this file](https://github.com/Orva-Studio/lite-query/blob/main/src/index.ts) and put it anywhere in your project.

2. If you'd like to install it [as a package](https://jsr.io/@orva/lite-query), you can use this command:

```bash
npx jsr add @orva/lite-query
```


## Features
Lite Query address all the data fetching issues raised in the [React Documentation](https://react.dev/reference/react/useEffect#what-are-good-alternatives-to-data-fetching-in-effects) caused by the useEffect hook such as:

	<Card title="Data Caching" icon="star">

	<Card title="Prefetching" icon="star">
